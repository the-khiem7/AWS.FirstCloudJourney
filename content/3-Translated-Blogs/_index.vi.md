Lưu trữ các bài blog AWS đã dịch và ghi chú liên quan để tham khảo nhanh trong kỳ thực tập.

### [Blog 1 - Mountpoint for Amazon S3 CSI driver v2: Hiệu năng cao và tối ưu tài nguyên cho Kubernetes](3.1-mountpoint-s3-csi-driver-v2/)

Bài blog giải thích driver Mountpoint for Amazon S3 CSI v2, cách chạy Mountpoint trong container không cần quyền root, chia sẻ cache giữa pod, tuân thủ SELinux và giảm chi phí so với mount qua host.

### [Blog 2 - SAP Load Testing: Cách tiếp cận Serverless với AWS](3.2-sap-load-testing-serverless/)

Hướng dẫn mô hình load testing SAP với kiến trúc serverless (Athena, CloudWatch, Grafana, Lambda) để kiểm thử hiệu năng mà không cần hạ tầng cố định.

### [Blog 3 - Phát trực tuyến trò chơi Windows bằng Proton 9 trên Amazon GameLift Streams](3.3-streaming-windows-games-proton9-gamelift-streams/)

Mô tả runtime Proton 9 trên GameLift Streams, dịch DirectX sang Vulkan để chạy game Windows trên Linux, hỗ trợ multi-tenancy giảm chi phí GPU và các lưu ý tương thích khi dùng Proton.
