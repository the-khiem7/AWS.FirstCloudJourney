## Week 4 Objectives:
- Deepen understanding of **AWS storage services**, including S3, Glacier, Snow Family, Storage Gateway, and AWS Backup.  
- Master S3 access mechanisms: Access Point, Storage Classes, ACL, CORS, Versioning, Replication, and performance optimization.  
- Perform end-to-end static website hosting with CloudFront CDN and public access configuration.  
- Learn the full lifecycle of migrating on-premises virtual machines to AWS using VM Import/Export.  
- Gain hands-on skills with Multi-AZ file systems, performance testing, deduplication, shadow copies, user quotas, and scaling operations.
